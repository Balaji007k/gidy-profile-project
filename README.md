# Full Stack Profile Management App

# gidy-profile-project
Full-stack Profile Page Replica built with React.js, Express.js, and MongoDB Atlas. Features profile viewing, editing, and skill endorsement with REST API integration.

## Tech Stack

Frontend:
- React.js
- Tailwind CSS
- Vite

Backend:
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication

## Features

- User Registration and Login
- JWT Authentication
- Profile Creation and Update
- Skill Endorsement
- Secure API
